Image search app utilizing Google Image Search API by Nikodem Ignas
type in any query into the URL in a form of
https://nidev866-image-search.herokuapp.com/api/imagesearch/*Your search here*
for example
https://nidev866-image-search.herokuapp.com/api/imagesearch/Dog images
You can optionally set custom offset such as
https://nidev866-image-search.herokuapp.com/api/imagesearch/*Your search here*?offset=2
You can also view most recent searches in my app
https://nidev866-image-search.herokuapp.com/api/latest/imagesearch
